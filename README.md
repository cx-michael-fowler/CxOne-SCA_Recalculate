Powershell Script to run recalculate on SCA scans marked for Recalculation. Includes the option to run on all SCA scans

Usage

Help

    .\CxOne - SCA_Recalculate.ps1 -help [<CommonParameters>]

Recalculate

    .\CxOne - SCA_Recalculate.ps1 [-all [daysSinceScan <int>]] [-silentLogin -apiKey <string>] [<CommonParameters>]
